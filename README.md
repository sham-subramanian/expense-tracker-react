💰 Expense Tracker (React)

A simple and responsive Expense Tracker built using React that helps users manage daily expenses and track their balance efficiently.

---

🚀 Features

- ➕ Add income and expenses
- ➖ Delete transactions
- 📊 Real-time balance calculation
- 📈 Income & Expense summary
- 💾 Data stored locally (localStorage)
- ⚡ Fast and responsive UI

---

🛠️ Tech Stack

- Frontend: React (Vite)
- Styling: CSS
- State Management: React Hooks (useState)
- Storage: Browser Local Storage

⚙️ Installation & Setup

1. Clone the repository

git clone https://github.com/sham-subramanian/expense-tracker-react.git

2. Navigate to project folder

cd expense-tracker-react

3. Install dependencies

npm install

4. Run the project

npm run dev
